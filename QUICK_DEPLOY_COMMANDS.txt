🚀 STEP 2: RAILWAY BACKEND

1. Go to: https://railway.app
2. Click "Start Project" → "Deploy from GitHub"
3. Select sporty-rabbi repo
4. Wait for auto-deploy to start
5. Once deployed:
   - Click "Variables" tab
   - Add: API_FOOTBALL_KEY = e55dfa2e957bf5f4c5f30d899f7212d6

6. Copy your PUBLIC DOMAIN URL (looks like: https://sporty-rabbi-backend.railway.app)

1. Open VS Code
2. Edit file: frontend/.env.production
3. Replace domain with YOUR Railway URL:

   VITE_WS_URL=wss://YOUR-RAILWAY-URL.railway.app

4. Save file

5. In Command Prompt:
   
   git add .
   git commit -m "Update production URLs"
   git push

⏱️  Wait 30 seconds
